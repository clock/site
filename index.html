# valorant shop checker

• every 24 hours your shop in the game Valorant resets.
every time the shop resets the program uses the riot api to
gather what's currently in my shop.

the information is then sent to a discord server through a python discord bot.

# software loader

• allows software owners to have some security for selling software.

• can be used for both .dll and exe files.

using comparing hardware components to the server's database, process hollowing
checking if dumping programs are open and anti debugging measures (ex heartbeat check)
we then stream bytes of the process to the user and manual map the bytes.

# arduino aimbot

• using an arduino to emulate a hid device for aimbot.

sends packets of the on-screen position of players through the serial port
then emulates a mouse and moves to said location.

# pcie dma

• using an FPGA that plugs into the PCIe slot of a desktop
you can read and write physical memory.

• works for both Windows, Linux, and macOS.

the FPGA is controlled by a second pc / laptop. you can interface
through a USB port and tell the FPGA were to read without restarting your main pc.

# anticheat

• client-side kernel anticheat for community csgo server.

checked for vuln drivers, heartbeat with the server, if a player is banned via hardware components
manual map detection, hooked functions return address, sig scan for known cheats, and verify loaded DLLs.
